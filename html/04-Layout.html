HTML 5 Structure Elements

Layout elements

<header> Defines the Masthead / title area - usually the first element on a page

<nav> Defines the Masthead / title area

<article> Defines self contains documents e..g blog posts, emails, comments

<footer> Defines the Footer area

<aside> Defines the Sidebar area, pull quotes, infromation is related by not part of main flow

<section> Defines the Generic area, or subject areas
